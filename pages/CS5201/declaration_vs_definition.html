Distinction between Declarations and definitions in C++

Defintion: Declaration (for C++ function)

Introduction of a name into a program

Defintion: Defintion (for C++ function)

A declaration that also associates something with the name

Note: Every Defintion is a Declaration but not vice versa

Some facts about C++ Declaration and Defintion syntax:

  • function prototypes are declarations only
  • functions with (possibly empty) bodies enclosed in braces is
    both a declaration and defintion
  • Most literature in C++ documentation specifically show functin
    defintions and do not worry about declarations
